 ## Como executar
 
 É necessário ter:
 - Python 3.9
 - Django 4.0.3
 
 ```bash
 git clone https://github.com/guilhermelbo/Todolist
 
 cd Todolist
 
 python manage.py migrate
 
 python manage.py runserver
 ```
