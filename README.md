 ## Como executar
 
 É necessário ter:
 - Python 3.9
 - Django 4.0.3
 
 ```bash
 git clone https://github.com/guilhermelbo/Todolist
 
 cd Todolist
 
 python manage.py runserver
 
 #Caso o comando acima dê algum erro, tente primeiro:
 python manage.py migrate
 ```
